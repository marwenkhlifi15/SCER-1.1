# Cherif Renewable Energy — Solar Solutions Website ⚡️

A responsive landing page for **Cherif Énergie Renouvelable**, a Tunisian company providing turnkey solar energy solutions for households and businesses.  
The website presents the company’s services, financing options, and contact information in a clean, modern interface.

---

## 🌞 Project Overview

This project is a static marketing website built to showcase:

- Complete **photovoltaic installation services** (from energy needs analysis to system sizing and inverter configuration).
- **Maintenance & performance monitoring** of solar installations.
- **Financing solutions**, including 0 TND upfront installation through the STEG program and support with subsidies and administrative procedures.

The goal is to provide a clear and trustworthy online presence for clients interested in transitioning to solar energy.

---

## 🛠️ Tech Stack

- **HTML5** – semantic structure for sections (hero, services, about, contact, etc.).
- **CSS3** – custom styling, layout, and responsiveness (no heavy framework).
- **JavaScript** (optional, if used) – basic interactivity (navigation, animations, etc.).
- **Responsive design** – optimized for desktop and mobile screens.

---

## ✨ Key Features

- 📌 **Hero section** presenting the company and main value proposition.  
- 🔧 **Services section** describing:
  - Complete solar panel installation (sizing, mounting, inverter setup).
  - Regular maintenance, cleaning, and performance checks.
  - Troubleshooting and repairs to keep systems efficient.
- 💶 **Financing & support**:
  - 0 TND upfront installation via STEG program.
  - Guidance on available grants and subsidies.
  - Help with administrative procedures for a smooth transition.
- 📝 **Informational content** explaining the benefits and reliability of solar energy solutions.
- 📱 **Responsive layout** so the site works well on phones, tablets, and desktops.
